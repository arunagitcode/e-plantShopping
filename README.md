# e-plantShopping

## Project Name: Paradise Nursery Shopping Application

**Overview:**  
The **e-plantShopping** project is a front-end React application for “Paradise Nursery,” allowing users to browse, select, and purchase houseplants. The app includes a **landing page**, a **product listing page**, and a **shopping cart page**. Users can add plants to the cart, adjust quantities, remove items, and view the total cost dynamically.  

**Features:**  
- Landing page with a background image, company name, description, and "Get Started" button.  
- Product listing page with **six or more plants**, grouped into **three categories**, each showing a thumbnail, name, price, and Add to Cart button.  
- Shopping cart page with **thumbnail, name, unit price, total price per plant**, quantity adjust buttons, delete button, checkout, and continue shopping functionality.  
- Dynamic cart icon in the header showing the total number of items.  
- State management using **Redux**.  
- Project deployed via **GitHub Pages**.
